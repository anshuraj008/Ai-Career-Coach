# Full Stack AI Career Coach with Next JS, Neon DB, Tailwind, Prisma, Inngest, Shadcn UI🔥🔥


## Setup

Before running the app, install the project packages first:

```bash
npm install
```

Then build and start the server:

```bash
npm run build
npm start
```

If you want to confirm the package scripts are available, check `package.json` first.

### Make sure to create a `.env` file with following variables -

```
DATABASE_URL=
NEXT_PUBLIC_CLERK_PUBLISHABLE_KEY=
CLERK_SECRET_KEY=

NEXT_PUBLIC_CLERK_SIGN_IN_URL=
NEXT_PUBLIC_CLERK_SIGN_UP_URL=
NEXT_PUBLIC_CLERK_SIGN_IN_FALLBACK_REDIRECT_URL=
NEXT_PUBLIC_CLERK_SIGN_UP_FALLBACK_REDIRECT_URL=

GEMINI_API_KEY=
```
